# MVP Scope

## Included

- Email and password auth.
- One user account with worker and employer modes.
- Profile editing for personal, worker and employer fields.
- Employer order publishing with address, coordinates, date and price.
- Worker nearby order search using the Haversine formula.
- Worker responses and employer assignment.
- Order start, completion, cancellation and dispute-ready status model.
- WebSocket chat for order participants.
- Mock wallet, escrow ledger and 10% commission from both sides.
- Reviews and rating recalculation.
- Minimal admin API for users, orders and transactions.

## Deferred

- Google and Apple OAuth.
- Real payment provider integration.
- Full web admin panel.
- Push notifications.
- Address geocoding and map provider integration.
